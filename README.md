# Clinica Medicala
„Gestionare Clinică Medicală” este un prototip interactiv realizat în Figma, destinat administrării activităților unei clinici.

Sistemul permite autentificarea utilizatorilor cu roluri diferite (admin și medic), gestionarea fișelor pacienților, adăugarea și modificarea acestora, precum și vizualizarea statisticilor privind pacienții și consultațiile.

Interfața este clară, coerentă și ușor de utilizat, iar navigarea între ecrane se realizează prin meniuri și butoane interactive.
